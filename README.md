# 🎧 Rhythm Insights: Spotify Analytics

## 📌 Project Overview

This project focuses on analyzing Spotify song data using SQL and building an interactive Power BI dashboard to visualize key insights. It helps understand music trends, artist performance, and user listening patterns.

---

## 🛠 Tools & Technologies Used

* MySQL (Data Analysis)
* Power BI (Data Visualization)
* CSV Dataset

---

## 📊 Key Features

* 🔝 Top 10 Most Streamed Songs
* 🎤 Artist Performance Analysis
* 📅 Year-wise Streaming Trends
* 🎧 Audio Features Analysis (Danceability, Energy, etc.)
* 🎛 Interactive Filters for better insights

---

## 🧠 SQL Analysis

* Performed 60+ SQL queries
* Used aggregation functions (SUM, AVG, COUNT)
* Applied GROUP BY, ORDER BY, and filtering conditions
* Extracted meaningful insights from raw dataset

---

## 📈 Power BI Dashboard

* Designed a one-page interactive dashboard
* Included KPI cards (Total Songs, Total Streams, Avg Streams)
* Used bar charts, line charts, and scatter plots
* Applied clean and modern UI design

---

## 📁 Project Files

* 📄 Dataset (spotify-2023.csv)
* 🧾 SQL Queries File
* 📊 Power BI Dashboard (.pbix)
* 🖼 Dashboard Screenshot

---

## 🚀 Outcome

This project demonstrates strong skills in data analysis and visualization by transforming raw Spotify data into actionable insights.
